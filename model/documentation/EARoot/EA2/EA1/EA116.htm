The BoreholeEventCollection is a collection of BoreholeEvents. Many collections can be managed within a single borehole. The grouping of BoreholeEvents in a collection is arbitrary and varies from one community to another.  Geologists tend to group lithologic observations in one collection - called “logs” - and samples in another one. The current model does not impose any grouping by (e.g. Borehole Log, Borehole Construction, etc.).<br/>Note : this class is an adaptation of ISO 19148 LE_Feature.<br/>
